## ğŸš€ Quick Start

### Prerequisites
- Java 8 or higher
- Gradle (included via wrapper)

### Building the Application
```bash
# Clone the repository
git clone <repository-url>
cd TDD-Banking

# Build the application
./gradlew clean build

# Run tests
./gradlew test

# Generate test coverage report
./gradlew jacocoTestReport
```

### Running the Application
```bash
# Run the application
java -cp build/classes/java/main banking.MasterControl

# Or run with JAR
java -jar build/libs/TDD-Banking-1.0-SNAPSHOT.jar
```

## ğŸ’¼ Core Features

### Account Management
- **Checking Accounts**: Daily banking with no restrictions
- **Savings Accounts**: Interest-bearing with withdrawal status tracking
- **CD Accounts**: Certificate of Deposit with compound interest

### Transaction Processing
- **Deposits**: Add money to any account
- **Withdrawals**: Remove money with overdraft protection
- **Transfers**: Move money between accounts

### Time Management
- **Interest Calculation**: Automated monthly interest calculation
- **Time Progression**: Advance time and apply interest
- **Fee Management**: Automatic fee application for low balances

### Command Processing
- **Input Validation**: Comprehensive command validation
- **Error Handling**: Graceful error handling and logging
- **Output Generation**: Formatted account status and transaction history
